# SkyWorld 3D Voxel Game

SkyWorld is a 3D voxel-based game with floating islands, block system, day/night cycle, and mobile support.

## Features
- 3D Voxel World with Floating Islands
- Block System (6 types: Grass, Stone, Wood, Iron, Diamond, Lava)
- Day/Night Cycle with Dynamic Lighting
- Inventory System
- Mobile Touch Controls
- Multi-language Support (Turkish/English)

## Live Demo
Visit the live version at: https://tropikkuslar-lab.github.io/SkyWorld/

## How to Play
- Use WASD keys to move
- Mouse to look around
- Click blocks to interact
- Use inventory system to manage blocks
- Toggle day/night cycle
